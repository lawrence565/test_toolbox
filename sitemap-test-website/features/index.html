Claude 核心功能

除了基本的文字生成與理解能力之外，Claude 提供多項進階功能，讓模型能夠處理更複雜的任務場景。這些功能大幅拓展了 AI 助手的應用邊界。

功能一覽

Extended Thinking

讓 Claude 在回答之前進行更深層的內部推理過程，適合數學證明、程式除錯、複雜邏輯分析等需要多步驟思考的任務。

Computer Use

Claude 可以直接操作電腦桌面環境，包括移動滑鼠、點擊按鈕、輸入文字與截取螢幕畫面，實現端到端的自動化工作流程。

Tool Use (Function Calling)

透過結構化的工具定義，Claude 可以呼叫外部 API、查詢資料庫、執行運算等操作，將 AI 能力無縫整合至既有系統。

整合方式

所有進階功能皆可透過 Anthropic Messages API 啟用。開發者只需在 API 請求中加入對應的參數設定即可。詳細的 API 規格請參閱 開放標準 區塊。
