Tool Use（工具呼叫）

Tool Use（又稱 Function Calling）讓 Claude 能夠呼叫開發者預先定義的外部工具，將 AI 的語言理解能力與實際系統操作結合起來。這使得 Claude 不僅能回答問題，更能實際執行任務。

運作方式

  1. 工具定義：開發者在 API 請求中以 JSON Schema 格式定義可用的工具，包含名稱、描述與輸入參數。
  2. 意圖辨識：Claude 分析使用者的請求，判斷是否需要呼叫工具以及該使用哪個工具。
  3. 參數生成：Claude 根據使用者的輸入，自動產生符合 Schema 的工具呼叫參數。
  4. 結果處理：開發者執行工具呼叫後，將結果回傳給 Claude，Claude 再根據結果生成最終回答。

工具定義範例

{
  "tools": [
    {
      "name": "get_weather",
      "description": "查詢指定城市的目前天氣狀況",
      "input_schema": {
        "type": "object",
        "properties": {
          "city": {
            "type": "string",
            "description": "城市名稱，例如：台北"
          }
        },
        "required": ["city"]
      }
    }
  ]
}

應用場景

  • 資料查詢：搜尋資料庫、查詢 API、取得即時資訊
  • 任務執行：發送郵件、建立工單、更新記錄
  • 計算處理：複雜數學運算、資料轉換、統計分析
  • 系統整合：連接 CRM、ERP、專案管理等企業內部系統
  • 多步驟工作流：將多個工具串接，完成端到端的自動化流程

與 MCP 的關係

Tool Use 是 Claude 內建的工具呼叫機制，而 Model Context Protocol (MCP) 則是 Anthropic 推動的開放標準，旨在標準化 AI 模型與外部工具之間的通訊協定。MCP 可以視為 Tool Use 的標準化延伸，讓工具定義可以跨平台共享與重用。

支援的模型

所有 Claude 模型均支援 Tool Use 功能：

  • Claude Opus
  • Claude Sonnet
  • Claude Haiku
