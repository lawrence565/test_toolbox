Extended Thinking（深度推理）

Extended Thinking 是 Claude 的一項進階功能，允許模型在生成最終回答之前，先進行一段內部的深度思考過程。這類似於人類在面對複雜問題時會先草擬思路、反覆推敲，最後才給出結論。

運作原理

當開發者在 API 請求中啟用 Extended Thinking 功能後，Claude 會：

  1. 先生成一段「思考過程」（thinking tokens），這段內容會呈現模型的推理步驟。
  2. 在思考過程中分解問題、考慮多種可能性、驗證中間結果。
  3. 基於完整的思考鏈，生成最終的高品質回答。

使用場景

  • 數學與邏輯推理：需要多步驟演算的數學題目、邏輯謎題。
  • 程式除錯：追蹤複雜程式碼的執行流程，找出潛在問題。
  • 策略分析：需要權衡多個方案優劣的決策場景。
  • 科學推論：根據已知條件進行假設驗證與推導。
  • 長文撰寫規劃：先組織大綱結構，再進行詳細寫作。

API 使用方式

透過 Anthropic Messages API，在請求參數中設定 thinking 欄位即可啟用： 

{
  "model": "claude-sonnet-4-20250514",
  "max_tokens": 16000,
  "thinking": {
    "type": "enabled",
    "budget_tokens": 10000
  },
  "messages": [...]
}

budget_tokens 參數控制模型可以使用的最大思考 token 數量。提高預算可以讓模型進行更深入的推理，但也會增加回應時間與成本。

支援的模型

  • Claude Opus — 搭配 Extended Thinking 可達到最強推理效果
  • Claude Sonnet — 日常使用的推薦組合
  • Claude Haiku — 輕量場景下的快速推理
