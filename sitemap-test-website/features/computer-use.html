Computer Use（電腦操作）

Computer Use 是 Claude 的一項突破性功能，使 AI 模型能夠直接與電腦桌面環境互動。Claude 可以觀看螢幕截圖、移動滑鼠游標、點擊按鈕、輸入文字，就像人類操作員一樣完成各種電腦任務。

運作原理

Computer Use 透過一個循環的感知—決策—行動迴路運作：

  1. 螢幕感知：系統定期截取螢幕畫面並傳送給 Claude 進行視覺分析。
  2. 決策規劃：Claude 理解目前的畫面狀態，決定下一步操作。
  3. 動作執行：Claude 發出具體的指令（移動滑鼠、點擊、輸入文字等），由環境代理執行。
  4. 結果驗證：再次截圖確認操作是否成功，若未完成則繼續下一步。

支援的操作

操作類型 說明
滑鼠移動 將游標移動到螢幕上的指定座標
點擊 左鍵、右鍵與雙擊操作
文字輸入 在任意文字欄位中輸入內容
鍵盤快捷鍵 使用 Ctrl+C、Ctrl+V 等組合鍵
螢幕截圖 擷取目前的螢幕畫面進行分析
拖曳操作 按住並拖動元素到目標位置

適用場景

  • 網頁表單自動填寫與提交
  • 跨應用程式的工作流自動化
  • 軟體 UI 測試與驗證
  • 資料從舊系統搬遷至新系統
  • 重複性桌面操作的自動化

安全考量

由於 Computer Use 賦予 AI 直接操控電腦的能力，Anthropic 建議在使用時注意以下安全措施：

  • 在隔離的虛擬環境或容器中執行
  • 限制網路存取範圍，避免存取敏感系統
  • 設定操作的允許清單（allowlist）
  • 建立人類監督與即時中斷機制

支援的模型

  • Claude Opus — 完整支援
  • Claude Sonnet — 完整支援
  • Claude Haiku — 尚未支援
