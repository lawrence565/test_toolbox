關於 Claude

Claude 是由 Anthropic 開發的大型語言模型系列。Anthropic 成立於 2021 年，致力於 AI 安全研究，其核心理念為「Constitutional AI」— 透過一組明確的價值準則來引導模型的行為，確保模型既有用又安全。

Claude 目前廣泛應用於程式開發、文件撰寫、資料分析、客戶服務自動化等場景，並透過 API、Claude.ai 網頁介面以及各大雲端平台（Amazon Bedrock、Google Cloud Vertex AI）對外提供服務。
