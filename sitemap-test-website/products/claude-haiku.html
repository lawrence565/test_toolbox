Claude Haiku

Claude Haiku 是 Claude 系列中最快速、最具成本效益的模型。它專為需要即時回應的高吞吐量場景而設計，在保持優秀品質的同時提供極低的延遲與費用。

核心特色

  • 極速回應：Claude 系列中延遲最低的模型，毫秒級的首 token 回應時間。
  • 超低成本：API 費用僅為 Opus 的數分之一，適合大規模部署。
  • 200K 上下文視窗：即使是輕量模型，也支援完整的長上下文處理。
  • 高吞吐量：支援大量並行請求，適合企業級批次處理需求。
  • Tool Use 支援：完整支援 工具呼叫 功能，可整合至自動化流程。

適用場景

  • 聊天機器人與客服自動化
  • 即時翻譯與內容摘要
  • 大量文件分類與標記
  • API 閘道的智慧路由判斷
  • 嵌入式 AI 功能（應用內建議、自動完成）

技術規格

項目 規格

Tool Use 支援 ✅

Computer Use —

與其他模型比較

Haiku 在速度與成本上遠優於 Sonnet 和 Opus，但在複雜推理任務上的表現不如後兩者。如果任務需要更深度的分析與創意生成，建議升級至 Sonnet 或 Opus。
